About System

Railways Cargo tracking system (RCTS)

Introduction

RCTS is a website having two main modules of Cargo Reservation and Cargo Tracking

Reservation

User have to sign-up to the RCTS for the Reservation of the consignment, User can select the type of item to be sent, quantity of item, Station from which consignment is to be send and Station where consignment is to be send. User can search for the cargo rails. User can Select the departure date and check the availability of the cargo rail on departure date. User can pay through the credit card and payment methods like UBL Omini, and EasyPesa.

Tracking

User can track down where the consignment has reached, in this way User have the full control on userâ€™s cargo consignment. Tracking can be done by the tracking number given to the user. Tracking tells the user the status of the consignment, consignment has reached to the destination or not, received or not, by whom it is received and date and time also.

Freight Rates

It gives the detail of rates of each item w.r.t the weight of item and distance.
